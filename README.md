A modern and responsive full-stack web application that allows users to browse, customize, and order bikes online. The system includes real-time price updates, cart management, secure checkout, and cloud-based deployment. Designed with clean UI/UX using Figma and built using industry-standard development practices.

 Features:

 1.Browse & Filter Bikes by type, size, and price

 2.Customize Models (colors, accessories, upgrades) with dynamic pricing

 3.Secure Checkout with shipping details & payment integration

 4.Order Confirmation via email

 5.Responsive UI (Desktop, Tablet, Mobile)

Tech Stack:

Frontend: HTML, CSS, JavaScript, React.js
Backend: Node.js, Express.js
Database: MySQL / PostgreSQL / MongoDB
API & Security: REST APIs, JWT Auth, SSL/TLS
Payments: Stripe / PayPal
Deployment: AWS EC2, AWS RDS, Heroku, Cloudflare CDN
Tools: Git, GitHub, GitHub Actions, Figma

 Project Purpose:

Built as a capstone project to demonstrate skills in full-stack development, UI/UX design, API integration, and cloud deployment, providing real-world e-commerce functionality.

Future Enhancements:

Real-time delivery tracking
